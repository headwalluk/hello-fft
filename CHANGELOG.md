# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2025-11-20

### Added
- Exit button in the main toolbar for convenient application closure.
- `.gitignore` file for proper repository hygiene.
- Cleaning instructions in README for build artifact management.

### Changed
- Updated README to use present tense for architecture description.
- Improved documentation clarity.

## [1.0.0] - 2025-11-20

### Added
- Initial release of hello-fft application.
- CSV reader functionality to parse input data.
- FFT processor using FFTW3 library.
- GUI implementation using gtkmm-4.0.
- Histogram widget for visualization.
- Unit tests for CsvReader and FftProcessor.
- CMake build system configuration.
